# Roadmap del Proyecto

# Introducción

El Roadmap define la evolución planificada del sitio web de Ingever Asociados S.A.S.

Su propósito es organizar el desarrollo por fases, establecer prioridades y servir como guía para futuras implementaciones.

No todas las funcionalidades deben desarrollarse al mismo tiempo. Cada fase debe completarse, probarse y documentarse antes de iniciar la siguiente.

# Deuda Técnica

Actualmente identificada:

- Completar responsive.
- Revisar consistencia visual.
- Optimizar imágenes.
- Revisar accesibilidad.
- Mejorar rendimiento.

La deuda técnica debe reducirse antes de incorporar nuevas funcionalidades importantes.

---

# Criterios de Finalización

Una fase se considera terminada cuando:

- El código funciona correctamente.
- No existen errores conocidos.
- Se realizaron pruebas.
- La documentación fue actualizada.
- El despliegue fue exitoso.

---

# Objetivo Final

Construir un sitio web corporativo moderno, escalable, accesible y fácil de mantener, que represente profesionalmente a Ingever Asociados S.A.S. y sirva como base para futuras ampliaciones.
